Environment setup
1. install Java Development kit on PC
2. After the installation is successfully completed check that JDK available.
(Rum command java -version in CMD)
3. Install environment for developing computer software - IntelliJ IDEA can be used.
4. After the installation is successfully completed
start IntelliJ IDEA, start new project, select a build automation tool (for ex. Maven)
and set path to SDK (path to previously installed JDK)
